
# 🍪 CookieWise

## Introduction

A smart browser extension that simplifies cookie consent by automatically detecting and analyzing cookie banners. CookieWise translates complex privacy policies into clear, actionable insights, giving users full control over their data.

## Problem Statement

- Low awareness in Pakistan about cookie banners and their purpose
- Most users accept cookies without understanding data collection risks
- Cookie policies are purposely written in complex, hard-to-understand language
- Personal data is silently tracked, stored, and shared with third parties
- Lack of transparency on where user data is being sent or used

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend logic & AI analysis |
| React | Frontend UI |
| Tailwind CSS | Styling |
| Selenium | Cookie banner detection & automation |

## Features

### Smart Automation
- Automatically detects cookie banners on any website
- One-click toggle (ON/OFF control)
- Notifies user if policy cannot be analyzed
- Simple and user-friendly interface

### Awareness & AI Analysis
- Shows website being analyzed
- Visualizes global data transmission
- Flags risky/suspicious clauses
- AI generates risk score & detailed insights

### User Control & Impact
- Default options: Accept All / Reject All / Custom
- Applies preferences automatically
- Saves time and effort for users
- Converts complex policies into clear insights

## Goal

Simplify and clarify privacy decisions, making users more aware and in control of their personal data.

## Prerequisites

- Python 3.8+
- Node.js (v14+)
- Chrome, MS Edge or Firefox browser
